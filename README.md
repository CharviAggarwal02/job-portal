# 📖 Overview

JobJunction is a modern recruitment platform developed using Flask that simplifies the hiring process for both employers and job seekers.

The application provides role-based dashboards, secure authentication, job posting and application management, resume uploads, and an administrative panel for managing the entire recruitment process.

The project demonstrates full-stack web development, backend architecture, authentication, database management, and role-based authorization.

---

# ❗ Problem Statement

Recruitment often involves multiple disconnected systems for job posting, candidate management, and application tracking.

Small organizations especially require an affordable centralized platform where recruiters can publish openings while candidates can easily search and apply for relevant jobs.

---

# 💡 Solution

JobJunction provides a complete recruitment management system where:

- Job seekers can create profiles and apply for jobs.
- Employers can publish and manage job listings.
- Administrators can monitor platform activity through a centralized dashboard.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/job-portal.git

cd job-portal
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

Windows

```bash
venv\Scripts\activate
```

Linux/macOS

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Setup Database

```bash
python setup.py
```

---

## Run Application

```bash
python run.py
```

Open

```
http://localhost:5000
```
